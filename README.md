# Idle Slayer Player

A simple bot to play Idle Slayer (Desktop).

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the dependencies.

```bash
pip install pyautogui pymsgbox asyncio pytesseract
```

## Usage

```python
python .\Player.py

```
